# 📅 Age Calculator

A simple Python program that calculates a person's age in different time units.

The program converts the user's age from **years** into:

* Months
* Days
* Hours
* Minutes

---

## 🚀 Features

* Simple command-line interface
* Converts age into multiple time units
* Beginner-friendly Python project
* Input validation

---

## ⚙️ How to Run

Make sure Python is installed on your system.

Run the program:

python age_calculator.py
